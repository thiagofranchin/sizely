# Contribuindo

## Requisitos

- Use a versão do Node definida em `/.nvmrc`.
- Use `npm` 10 ou superior.

## Fluxo mínimo

1. Crie uma branch a partir da principal.
2. Instale dependências com `npm install`.
3. Faça mudanças pequenas e objetivas.
4. Rode `npm run lint` e `npm run test`.
5. Abra um pull request com contexto, motivação e impacto da mudança.

## Padrões

- Preserve o comportamento client-side do projeto.
- Evite introduzir dependências sem necessidade clara.
- Mantenha a interface consistente com o design já estabelecido.
- Inclua testes para mudanças em regras de negócio sempre que possível.
