# Política de Segurança

Se você encontrar uma vulnerabilidade de segurança no Sizely, não abra uma issue pública.

Reporte de forma privada por um destes canais:

- GitHub Security Advisory do repositório `thiagofranchin/sizely`
- Email do mantenedor, se disponibilizado no perfil do projeto

Ao reportar, inclua:

- descrição do problema;
- impacto estimado;
- passos para reproduzir;
- versão ou commit afetado;
- sugestão de correção, se houver.

Farei uma análise inicial assim que possível e, quando confirmado, a correção será tratada antes da divulgação pública dos detalhes.
